Easy Leave Management System
📌 Project Overview

The Easy Leave Management System is a web-based application designed to simplify and automate the leave request and approval process within an organization or institution. The system helps employees/students apply for leave online while allowing administrators or managers to review, approve, or reject requests efficiently.

This project reduces paperwork, improves transparency, and maintains proper leave records digitally.

🚀 Features
User Registration & Login
Apply for Leave Online
View Leave History
Leave Approval/Rejection by Admin
Leave Status Tracking
Admin Dashboard
Employee/Student Record Management
Secure Authentication System
Database Integration for Record Storage
